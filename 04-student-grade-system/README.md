# 🎓 Student Grade System

This mini project checks each student's marks and assigns a grade based on their score.

## 👨‍🎓 Logic

- If marks >= 90 → Grade A 🔥
- If marks >= 75 → Grade B 💪
- If marks >= 50 → Grade C ✅
- Otherwise → Grade F ❌
